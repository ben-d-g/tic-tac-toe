# tic-tac-toe

Assignment: Build a tic-tac-toe game on the command line where two human players can play against each other and the board is displayed in between turns.

## Classes:

- Game
- Player
- Grid
- Cell

## Cell

Variables:

- Value - `"X"`, `"O"` or `"-"`

Methods:

- Get value
- Set value

## Grid

Variables:

- grid - 3 by 3 2d array of Cell objects

Methods:

- Get cell
- Set Cell
- Check for winner
- Get Grid

## Player

Variables:

- Name

Methods:

- Make move

## Game

Variables:

- Grid
- Player 1
- Player 2
- Done?

Methods:

- Display Grid
